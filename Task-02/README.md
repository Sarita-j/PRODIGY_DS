# Titanic Dataset - Data Cleaning and Exploratory Data Analysis (EDA)

### Task 02

This project performs Data Cleaning and Exploratory Data Analysis (EDA) on the Titanic dataset using Python.

---

## Objective

- Load the Titanic dataset
- Clean missing values
- Explore relationships between variables
- Visualize important patterns using graphs

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## Data Cleaning

The following preprocessing steps were performed:

- Filled missing Age values using Median
- Filled missing Embarked values using Mode
- Removed Cabin column due to excessive missing values

---

## Visualizations

The project includes:

- Survival Count
- Gender Distribution
- Survival by Gender
- Passenger Class Distribution
- Survival by Passenger Class
- Age Distribution
- Fare Distribution
- Correlation Heatmap
- Age vs Survival
- Fare vs Passenger Class
- Pair Plot

---

## Key Findings

- Female passengers had higher survival rates.
- First-class passengers survived more often.
- Most passengers were between 20–40 years old.
- Fare distribution is highly right-skewed.
- Higher fare passengers generally had better survival chances.
